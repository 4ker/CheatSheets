# Bash Cheat Sheet
### Source a file
`source [file/path]`  

### Movements
Beginning-of-line `<C-a>`  
End-of-line `<C-e>`  
Forward-char `<C-f>`  
Backward-char `<C-b>`

### Corrections
Delete back one word `<C-w>`  
Delete back to start of line `<C-u>`  

### Hex Editor
`xxd <filename>`  

### Clear current terminal history 
`history -c`
